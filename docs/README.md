# Mitosis Mike Docs

Mitosis Mike is a submission for the `$5K GameName Game Jam` built around the theme `Invincible`.

The central idea is simple: Mike cannot be permanently stopped. When he is harmed, he divides instead of dying. Each split leaves him smaller, creates more controllable bodies, and turns enemy attacks and level hazards into puzzle tools instead of pure failure states.

## High-Level Pitch

Mitosis Mike is a puzzle-platform game set in an underground laboratory complex. Mike, a green laboratory test subject, escapes confinement and discovers that damage causes mitosis instead of death. The player solves levels by weaponizing that strange resilience: getting hit in the right place, changing size, splitting into multiple bodies, and using environmental threats as part of the solution.

## Current Design Snapshot

- Core fantasy: an invincible protagonist whose setbacks create new tactical options.
- Core interaction: splitting Mike into smaller pieces to access routes and solve spatial puzzles.
- Current implemented gameplay direction: smaller Mike bodies fit through tighter spaces.
- Planned expansion: size-dependent jumping, momentum changes, elemental interactions, and potion effects.
- Level goal: feed enough of Mike's mass into a Recombobulator to reassemble him and open the way forward.
